WEB

웹(Web)은 인터넷을 통해 정보와 서비스를 제공하는 시스템으로, 전 세계 사람들이 다양한 정보에 접근하고 상호작용할 수 있게 하는 중요한 도구입니다.
웹의 주요 구성 요소와 기능은 다음과 같습니다.

웹의 주요 구성 요소

  1. 웹 페이지(Web Page):
    2. 개별적인 문서로, HTML(HyperText Markup Language)로 작성됩니다. 텍스트, 이미지, 비디오, 링크 등 다양한 멀티미디어 콘텐츠를 포함할 수 있습니다. 웹 페이지는 웹 브라우저에서 렌더링되어 사용자에게 표시됩니다.
  2. 웹 사이트(Web Site):
    3. 관련된 웹 페이지들의 모음입니다. 특정 도메인 이름(예: www.example.com)으로 식별됩니다. 회사, 조직, 개인 등이 정보를 제공하거나 서비스를 제공하는 데 사용됩니다.
  3. 웹 브라우저(Web Browser):
    4. 사용자가 웹 페이지에 접근하고 볼 수 있도록 해주는 소프트웨어 애플리케이션입니다. 대표적인 웹 브라우저로는 Google Chrome, Mozilla Firefox, Microsoft Edge, Apple Safari 등이 있습니다. HTML, CSS, JavaScript 등을 해석하여 웹 페이지를 표시합니다.
  4. 웹 서버(Web Server):
    5. 웹 페이지를 저장하고 사용자 요청에 따라 이를 제공하는 컴퓨터 시스템입니다. 사용자가 브라우저에서 URL을 입력하면, 해당 URL에 해당하는 웹 서버가 요청을 받아 웹 페이지를 전송합니다.
  5. URL(Uniform Resource Locator):
    6. 웹 페이지의 주소를 나타내는 고유 식별자입니다. 예: https://www.example.com/page1.html

웹의 주요 기능

  1. 정보 공유:
    2. 웹은 전 세계 어디서든 정보에 접근할 수 있게 해줍니다. 학술 자료, 뉴스, 블로그, 백과사전 등 다양한 형태의 정보를 제공합니다.
  2. 커뮤니케이션:
    3. 이메일, 소셜 네트워크, 포럼, 채팅 등 다양한 방식으로 사람들 간의 소통을 가능하게 합니다. 비즈니스, 교육, 사회적 활동에서 중요한 역할을 합니다.
  3. 전자 상거래:
    4. 웹을 통해 상품과 서비스를 구매하고 판매할 수 있습니다. 온라인 쇼핑몰, 결제 시스템, 디지털 마켓플레이스 등이 포함됩니다.
  4. 엔터테인먼트:
    5. 음악, 영화, 게임, 스트리밍 서비스 등 다양한 엔터테인먼트 콘텐츠를 제공합니다. 사용자가 다양한 형태의 디지털 콘텐츠를 즐길 수 있게 합니다.
  5. 교육:
    6. 웹은 온라인 강의, 교육 자료, 학습 플랫폼 등을 통해 학습 기회를 제공합니다. 전 세계 어디서나 교육에 접근할 수 있게 합니다.

웹의 동작 원리

클라이언트-서버 모델: 사용자가 웹 브라우저를 통해 웹 페이지를 요청하면, 브라우저는 해당 요청을 웹 서버로 전송합니다.
웹 서버는 요청받은 페이지를 찾아 클라이언트(브라우저)에게 응답으로 전송합니다.
브라우저는 응답받은 페이지를 사용자에게 렌더링하여 표시합니다.
HTTP/HTTPS: 웹 페이지 요청과 응답은 HTTP(HyperText Transfer Protocol) 또는 HTTPS(HyperText Transfer Protocol Secure)를 통해 이루어집니다.
HTTPS는 데이터를 암호화하여 보안성을 높인 프로토콜입니다.
