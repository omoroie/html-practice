CSS

CSS란 무엇인가요?
CSS(Cascading Style Sheets)**는 HTML이나 XML(예: SVG, MathML, XHTML)로 작성된 문서의 스타일을 기술하는 스타일시트 언어입니다. CSS는 문서가 화면, 종이, 음성 또는 기타 매체에 어떻게 렌더링되어야 하는지를 설명합니다. CSS의 주요 특징

1. 내용과 표현의 분리:

CSS는 문서의 내용(HTML이나 유사한 마크업 언어로 작성된 것)과 문서의 표현(CSS로 작성된 것)을 분리할 수 있게 합니다. 이러한 분리는 다음과 같은 장점을 제공합니다: 내용 접근성 향상 표현 특성 지정에 더 많은 유연성과 제어 제공 여러 페이지가 동일한 스타일을 공유 가능 구조적 콘텐츠의 복잡성과 반복 감소

2. 선택자와 속성:

CSS는 선택자를 사용하여 문서의 특정 요소에 스타일을 적용합니다. 선택자는 요소의 타입, 클래스, ID, 속성 등을 기반으로 할 수 있습니다. 각 선택자는 하나 이상의 요소를 타겟팅하며, 스타일은 속성의 집합을 통해 적용됩니다.

3. 계단식 우선 순위(Cascading Priority):

CSS는 여러 스타일 규칙이 동일한 요소에 적용될 때, 어떤 규칙이 우선하는지를 결정하기 위해 계단식 우선 순위를 사용합니다. 우선 순위는 다음과 같은 규칙을 따릅니다: 인라인 스타일(요소 내에 직접 지정된 스타일) ID 선택자 클래스, 속성 및 가상 클래스 선택자 요소 선택자 상속된 스타일

4. 반응형 디자인:

CSS는 미디어 쿼리를 사용하여 다양한 디바이스와 화면 크기에 대응하는 반응형 웹 디자인을 가능하게 합니다. 이를 통해 동일한 HTML 구조를 사용하면서도 다양한 디바이스에 적합한 레이아웃과 스타일을 적용할 수 있습니다.
